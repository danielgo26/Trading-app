Short project documentation:

- Backend(Spring Boot) & Frontend(React) part

- Using Kraken API for fetching 20 crypto currencies in real time using WebSockets. The project uses a configure file -
  supported_crypto_pairs.csv, located in the resources foulder in src. Its main idea is to set the 20 crypto currencies names, 
  symbols and currency signs that next to be fetched from the Kraken API.

- The project uses PostgreSQL as database provider. The project is built without using the Object-relational mapping (ORM) model
  with the main purpose of writing plain SQL queries. The ones used in the project can be found in the SQL_Queries.txt file.

- The applied images and screen recording demonstrate a regular user experience in the system.
